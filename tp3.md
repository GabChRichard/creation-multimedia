# TP3 – Montage vidéo court sur Canva  
**(20% de la note finale)**

## CONSIGNE

Vous êtes approchés pour réaliser un montage vidéo court qui présente votre plus grande ambition (votre perspective d'avenir, votre rêve le plus fou, etc.). Le contenu doit demeurer approprié à un cadre scolaire.

Le projet doit comprendre les demandes suivantes :

1. Préproduction  
2. Montage vidéo  
3. Livrable (remise)

## Préproduction

La préproduction comprend un texte au format Word avec les éléments suivants  :

- Une description de votre plus grande ambition?
- Une description du ton (humoristique, dramatique, profesionnel, techno, etc.)   
- Une description visuelle (quelles type d'images seront montrées?)
- Une description sonore (quelles type de musique sera utilisée?)

## Montage vidéo

Le montage doit comprendre les éléments suivants :

- Une durée entre **30 secondes et 1 minute**
- L'intégration d'une bande sonore (musique et/ou effets sonores)
- Un titre animé (title reveal)
- Un générique de fin avec les éléments suivants :
  - Le nom du réalisateur / de la réalisatrice
  - Le nom du cours avec le superviseur (enseignant)
  - La liste complète des sources pour les images et le son
  - Les noms en remerciements

## Livrable

Le tout doit être remis dans un dossier à votre nom avec les éléments suivants :

- Le Fichier texte d'intention
- Le montage final en format `.mp4`
- Le fichier de travail Canva (lien de partage)

## Consignes additionnelles

- Travail individuel  
- Les images sous copyright sont permises — **aucune image générée par IA**  
- Présentation en classe lors du **cours 11 (13 avril 2026)**

# CRITÈRES D'ÉVALUATION

- Clarté de l'intention — /5

- Clarté des indications sonores — /5  

## Montage vidéo — /75

- Pertinence des choix sonores — /20  
- Fluidité des animations et transitions — /15 

## Livrables — /10

- Respect du ton et de l'intention — /5  
- Conformité des éléments — /5  

## Total — /100
